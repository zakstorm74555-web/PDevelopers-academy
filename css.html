Colors & Selectors

Selectors point to the HTML element. Colors can be defined by name, HEX, or RGB.

/* Tag Selector */
p { color: #D4AF37; } 

/* Class Selector (Best for reuse) */
.btn-gold { background-color: gold; }

/* ID Selector (Unique) */
#main-banner { background: rgb(0, 0, 0); }
The Box Model

Every element in CSS is a box. Understanding Padding (Inside space) and Margin (Outside space) is vital.

Box Demo 
.box {
  width: 300px;
  padding: 20px;  /* Space between text and border */
  border: 5px solid white;
  margin: 40px;   /* Space outside the box */
  box-sizing: border-box; /* Keeps width fixed */
}
Measurements (Units)
  • PX: Fixed pixels (Best for borders).
  • %: Relative to parent width (Best for banners).
  • REM: Relative to root font size (Best for Text).
  • VH/VW: Viewport height/width.
h1 { font-size: 2.5rem; }
.hero { height: 100vh; width: 100%; }
Flexbox Mastery

Flexbox is how we align elements horizontally or vertically without complex math.

.container {
  display: flex;
  justify-content: center; /* Horizontal alignment */
  align-items: center;     /* Vertical alignment */
  flex-direction: row;     /* column for vertical stack */
}
Elite Theme Variables

Professional devs use :root to store theme colors. This makes changing the whole site's theme a 1-second job.

:root {
  --main-color: #004D40;
  --text-color: #ffffff;
}

body {
  background-color: var(--main-color);
  color: var(--text-color);
}
