PHP Engine

PHP: Hypertext Preprocessor. The engine that powers 80% of the web, including Facebook and WordPress.

1. Client vs Server Side

Unlike HTML/CSS which run in the browser, PHP runs on the Server (Apache/Nginx). The browser never sees your PHP code, only the HTML result.

<?php
  // Browser will never see this comment
  echo "<h1>Hello from the Server!</h1>";
?>
2. Data Handling: $_POST & $_GET

This is how we collect data from HTML forms. $_POST is secure for passwords, while $_GET shows data in the URL.

// Collecting data from a login form
$username = $_POST['user'];
$password = $_POST['pass'];

echo "Welcome back, " . $username;
3. Connecting to MySQL (The Heart)

A static site becomes a "Web App" when it connects to a database. We use PDO because it's more secure.

$host = 'localhost';
$db   = 'academy_db';
$user = 'root';
$pass = '';

$pdo = new PDO("mysql:host=$host;dbname=$db", $user, $pass);
echo "Connected to Jhelum Database!";
4. Security: The Golden Rule

Never trust user input. Hackers use SQL Injection to steal data. Always use Prepared Statements.

CRITICAL: Never write: "SELECT * FROM users WHERE id = $id". Always use placeholders. 
// The Safe Way (Prepared Statements)
$stmt = $pdo->prepare('SELECT * FROM users WHERE email = ?');
$stmt->execute([$email]);
$user = $stmt->fetch();
