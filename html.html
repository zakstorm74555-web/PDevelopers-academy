1. The Head & Meta Tags

Meta tags provide metadata about the HTML document. They are invisible to users but critical for search engines (SEO).

<head>
  <!-- Character encoding -->
  <meta charset="UTF-8">
  <!-- Responsive Viewport -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <!-- SEO Description -->
  <meta name="description" content="Learn Coding at PDevelopers">
</head>

2. Header & Banner

The <header> contains the logo and navigation. A "Banner" is usually a <section> or <div> with a background image.

PDEVELOPERS ACADEMY BANNER
<header>
  <h1>Logo</h1>
  <nav> ... </nav>
</header>

<section class="banner">
  <h2>Welcome to the Future</h2>
</section>

3. The Mighty <div>

The <div> is a generic container. It's the most used tag to group elements for styling with CSS.

<div class="container">
  <div class="card">
    <p>This is inside a division block.</p>
  </div>
</div>

4. Images & Graphics

Use <img> to embed photos. Always use the alt attribute for accessibility.

<img src="icon.png" alt="PDevelopers Logo" width="150" height="150">

5. Buttons & Links (<a>)

Links (<a>) navigate to URLs, while <button> tags are used for actions (like submitting forms).

<!-- Hyperlink -->
<a href="https://google.com" target="_blank">Open Link</a>

<!-- Button -->
<button type="button" onclick="alert('Hello')">Click Me!</button>
